# APP程式設計期末專案

一個簡易的記帳本，首先要進行註冊再登入
登入後可以看到新增款項以及下方目前款項的頁面，輸入日期、品項、金額、備註後按下新增即可新增款項
且目前款項會出現剛剛新增的款項，款項旁邊會有編輯按鈕(鉛筆圖案)和刪除按鈕(垃圾桶圖案)
點擊編輯按鈕即可對其進行修改，點擊刪除則會立馬刪除此款項
